## Dev Garage

Small shell tools for daily development and ops. All scripts use the `dg_` prefix.

### Scripts

| Script | Description |
|--------|-------------|
| `dg_check_hardware.sh` | System hardware specs (CPU, RAM, disk, GPU, network) |
| `dg_check_image.sh` | Detect image file format by magic bytes |
| `dg_check_network.sh` | Check internet access & proxy/VPN status |
| `dg_domain_lookup.sh <domain>` | DNS lookup via `dig` |
| `dg_reverse_lookup.sh <ip>` | Reverse DNS lookup |
| `dg_whois_lookup.sh <domain>` | WHOIS query |
| `dg_find_port.sh <port>` | Find which process occupies a port |
